# Tidal

Assuming all prerequisites have been installed:

[Install instructions: docs/tidal/](/docs/tidal)

To get started, run dirt: 

```
cd vendor/dirt && make clean; make
./vendor/dirt &
```

Launch your editor of choice:

```
emacs core.tidal
```
